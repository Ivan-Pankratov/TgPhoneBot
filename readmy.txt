Решена задача Телефонный справочник в виде Телеграмм Бота
    с внешним хранением информации в виде файла Базы Данных 'phone.db'
* Управление Ботом осуществяется через команды.
    Доступные команды описаны в файле 'manual.txt'
    или вызываются командой /help
* Управление данными осуществляется через
    SQL- запросы из дополнительного файла 'phone_metods.py'.
* После выставления оценки Токен Бота будет удалён.
* Со справочником доступны действия:
    + создание файла базы данных при её отсутствии
    + просмотр всего справочника, или сообщение о пустом списке
    + поиск по фамилии с проверкой на пустую команду
    + добавление записи с проверкой на пустую или неполную команду
    + удаление записи по фамилии с проверкой на пустую команду
    + сортировка списка по одному любому из парамертов
    + очистка списка полностью
    + удаление файла справочника
    + вывод справки
