Список команд справичника:
/help - вызов справки (этого файла);
/start - создание справочника;
/all  - вывод текущего списка контактов;
/add Фамилия Имя Отчество 123456 - добавление контакта в текущий список (образец: Через пробел и номер одним числом);
/delete Фамилия - удаление контакта из справочника по фамилии;
/find Фамилия - поиск контакта по фамилии;
/sort Фамилия/Имя/Отчество/Номер - Сотрировка по одному из параметров;
/clear - очистка всех записей справочника;
/oops - удаление справочника совсем.
